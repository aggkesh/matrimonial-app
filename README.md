# Matrimonial App
Matrimonial Android App

This GitHub repository contains the code of Matrimonial Android app. 
Matrimonial is a demo application based on [RandomUser.me] API that shows the usage of 
- MVVM Pattern
- Picasso
- Retrofit
- Room Database
